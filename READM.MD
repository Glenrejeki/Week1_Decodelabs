# Link-in-Bio Clone (DecodeLabs Project 1)

A mobile-first personal link tree website built with pure HTML5, CSS3, and JavaScript.

## Features
- **Mobile-First Design**: Optimized for mobile screens with responsive layout for desktop.
- **DecodeLabs Design System**:
  - Background: Moonlit Grey (#F2F0EA)
  - Buttons: Mocha Mousse (#A5856F)
  - Hover Effect: Ethereal Blue (#A0D4E0)
- **Interactivity**:
  - Copy to Clipboard: Easily share your profile link.
  - Web Share API: Native sharing experience on mobile devices.

## Tech Stack
- HTML5 (Semantic Elements)
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6+, Web APIs)

## Deployment to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root.
3. Follow the prompts to deploy.

Alternatively, push this code to a GitHub repository and connect it to Vercel for automatic deployments.
